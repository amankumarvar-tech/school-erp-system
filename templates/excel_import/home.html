📋 Import Rules & Tips

✅ Kya karna hai:

  • Template download karke usme data bharo
  • Column headers mat badlo
  • Date format YYYY-MM-DD use karo (e.g. 2010-05-15)
  • Pehle Students import karo, phir Fees
  • Existing records update ho jaate hain (Admission No / Employee ID se)

❌ Kya nahi karna:

  • Header row delete mat karo
  • Column order mat badlo
  • Required (*) fields blank mat chhodna
  • Fee import se pehle student must exist karna
  • .csv file mat upload karo — sirf .xlsx
